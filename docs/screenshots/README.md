# Screenshots Guide

Add real screenshots with these filenames so docs stay consistent:

- `01-gateway-status-ok.png` — healthy `openclaw gateway status`
- `02-security-audit-summary.png` — `openclaw security audit --deep` result
- `03-dashboard-localhost.png` — dashboard at `http://127.0.0.1:18789/`
- `04-token-mismatch-example.png` — sample unauthorized/token mismatch case
- `05-recovery-success.png` — post-fix healthy status

Tips:
- Blur/redact any tokens, usernames, or host-specific secrets.
- Keep resolution readable (at least 1280x720).
- Prefer terminal screenshots with full command + output visible.
